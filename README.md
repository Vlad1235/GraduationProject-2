Программа для получения GET запроса о погоде в городе Челябинск.

Использование в проекте технологий:
1. SpringBoot
2. Spring Rest
3. Spring Aop
4. Liquibase
5. Hibernate

API для запроса
http://localhost:8080/Chelyabinsk/all
http://localhost:8080/Chelyabinsk/{wheatherId}
